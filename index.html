HTML Tables

What is HTML Tables?

HTML Tables allow web developers to arrange data into rows and columns.
They can contain all sorts of HTML elements; text, images, lists, other tables, etc.

HTML Table Tags:

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The sets of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Tabel Foot The set of rows defining the footer in a table.

CSS Specificity

What is CSS Specificity?

CSS Specificity is the means by which browsers decide which CSS property values are the most relevant to an element and, therefore, will be applied. Specificity is based on the matching rules which are composed of different sorts of CSS selectors.

Selector Types

The following list of selector types increases by specificity:

  1. ID selectors
  2. CLASS selectors
  3. TYPE selectors

ID selectors matches elements by node name. In other words, it selects all elements of the given type within a document.

Class selectors matches elements based on the contents of their class attribute.
